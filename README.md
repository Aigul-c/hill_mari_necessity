# hill_mari_necessity
Example database for my article 
Zakirova, Aigul. To appear. Necessity constructions in Hill Mari: a diachronic corpus perspective. Journal of Uralic Linguistics (2) 2026

The database includes examples from 1890-1910s and 2010s expressing necessity, as well as those that feature the same/related markers but express other modal or non-modal meanings. This allows exploration of the diachronic relations between various constructions and comparison across related constructions.
